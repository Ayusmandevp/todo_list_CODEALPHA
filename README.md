# Ultimate To-Do List Web App

A modern and responsive To-Do List web application built using HTML, CSS, and JavaScript.

This project helps users manage daily tasks efficiently with advanced features like task categories, due dates, drag & drop, dark/light mode, search functionality, and browser localStorage support.

# Features

## Task Management
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed

## Advanced Features
- Task categories
- Due dates
- Priority levels
  - Low
  - Medium
  - High

- Drag & Drop task reordering
- Search tasks instantly
- Dark / Light theme toggle
- Persistent storage using localStorage

## User Experience
- Responsive design
- Smooth animations
- Mobile friendly
- Keyboard support (Press Enter)
- Modern glassmorphism UI
